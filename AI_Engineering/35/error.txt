                ## ЁЯФО What `fs` means
                - **`fs` = File System**
                - ItтАЩs a **core module in Node.js** that lets you interact with your computerтАЩs file system.
                - With `fs`, you can:
                - Read files (`fs.readFile`)
                - Write files (`fs.writeFile`)
                - Create or delete directories
                - Check if files exist
                - Stream file contents

                ---

                ## Two styles of using `fs`
                1. **Callback-based API (older style):**
                ```js
                const fs = require("fs");
                fs.readFile("podcasts.txt", "utf-8", (err, data) => {
                    if (err) throw err;
                    console.log(data);
                });
                ```

                2. **Promise-based API (modern style):**
                ```js
                import fs from "fs/promises";
                const text = await fs.readFile("podcasts.txt", "utf-8");
                console.log(text);
                ```

                The second style (`fs/promises`) is what we used in your project, because it works seamlessly with `async/await`.
