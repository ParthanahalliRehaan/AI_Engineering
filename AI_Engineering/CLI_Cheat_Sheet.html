Navigation & Directories Basics
Move around the filesystem quickly. Tip: use cd - to jump back
Command What it does Example
pwd Print current working directory path. pwd
cd Change directory. Use cd ~ for home, cd .. for parent, cd - to go back. cd /var/log
ls List files and directories. -l for details, -a to show hidden. ls -la
mkdir Create a directory. -p to create parent paths. mkdir -p src/routes
rmdir Remove an empty directory. rmdir old_logs
Files & Content Core
Create, move, delete and view files. Use less for big logs
Command What it does Example
touch Create an empty file or update its timestamp. touch index.js
cp Copy file or directory. Use -r for recursive. cp -r src/ backup/src/
mv Move or rename files/directories. mv old.txt new.txt
rm careful Remove files. Use -r for directories and -i for interactive safety. rm -ri temp/
ln -s Create symbolic (soft) link to a file or directory. ln -s /var/log/nginx access-logs
cat Print file contents to the terminal. cat config.json
less View file interactively (scroll, search). less -SN app.log
head / tail Show the first or last lines of a file. Use -n to set count. tail -n 50 server.log
File Formats & Text Transform
Change encodings, line endings and text structure. Line endings & encodings
Command What it does Example
dos2unix Convert Windows CRLF line endings to Unix LF in text files. dos2unix script.sh
unix2dos Convert Unix LF line endings to Windows CRLF. unix2dos notes.txt
iconv Convert character encoding of a file. iconv -f ISO-8859-1 -t UTF-8 in.txt -o out.txt
tr Translate or delete characters from input. tr '[:lower:]' '[:upper:]' < names.txt
sed Stream editor for search, replace and transformations. sed 's/old/new/g' file.txt
awk Pattern scanning and processing (columns, filters). awk '{print $1,$3}' data.txt
cut Extract fields/columns from text. cut -d':' -f1 /etc/passwd
rename Batch rename files using regex. rename 's/.log/.old/' *.log
file Detect file type and encoding information. file README.md
Permissions & Ownership Security
Control who can read, write and execute. r=4, w=2, x=1
Command What it does Example
chmod Change file permissions using octal or symbolic notation. chmod 755 script.sh, chmod +x run.sh
chown Change file owner and group. chown user:group app.log
id Display current user and group IDs. id
umask Show or set default permission mask for new files. umask 022
Search & Find Discovery
Find files and text quickly. Use grep -rn for code
Command What it does Example
find Search for files and directories by name, size, etc. find . -name "*.js"
grep Search for patterns inside files. grep -rn "TODO" src/
locate Quickly find files by name using a cached index. locate nginx.conf
which Show path of an executable in $PATH. which node
whereis Locate binary, source and man page. whereis git
whatis One-line description of a command. whatis ssh
Pipes & Redirection Compose
Chain commands and control input/output. | connects commands
Syntax What it does Example
cmd1 | cmd2 Send output of cmd1 as input to cmd2. ls | grep ".log"
>, >> Redirect stdout to file (overwrite / append). echo "hi" > file.txt
2> Redirect stderr to file. command 2> errors.log
< Use file as stdin for a command. sort < unsorted.txt
xargs Build and execute command lines from stdin. find . -name "*.tmp" | xargs rm
tee Write output to file and screen simultaneously. ls | tee listing.txt
&&, ||, ; Conditional and sequential command chaining. mkdir build && cd build
Processes & Jobs Monitoring
Inspect and control running programs. Use Ctrl+Z for jobs
Command What it does Example
ps aux Show all running processes with details. ps aux | grep node
top / htop Interactive process viewers. top -o%MEM
kill Send signal to a process by PID (default: TERM). kill 1234
killall Kill processes by name. killall node
jobs List background jobs for current shell. jobs
bg / fg Resume jobs in background / foreground. bg %1, fg %1
nohup Run a command immune to hangups. nohup node server.js &
time Measure how long a command takes to run. time npm test
Networking Remote
Test connectivity and talk to servers. Essential for backend dev
Command What it does Example
ping Test network reachability and latency. ping google.com
curl Transfer data to/from URLs, test APIs. curl -X GET http://localhost:3000/health
wget Download files via HTTP/HTTPS. wget -c https://example.com/file.tar.gz
ssh Secure shell into a remote machine. ssh user@server
scp Secure copy files between hosts over SSH. scp app.tar.gz user@server:/opt
ip a Show network interfaces and addresses. ip a
ss -tuln Show listening sockets and ports. ss -tuln
traceroute Trace the route packets take to a host. traceroute example.com
nslookup Query DNS records for a domain. nslookup example.com
Compression & Archives Packaging
Pack and unpack project files. Use for backups & deploys
Command What it does Example
tar -cvf Create a tar archive from files or directories. tar -cvf app.tar src/ package.json
tar -xvf Extract a tar archive. tar -xvf app.tar
tar -xvzf Extract a compressed .tar.gz archive. tar -xvzf app.tar.gz
zip Create a zip archive. zip -r app.zip src/ package.json
unzip Extract a zip archive. unzip app.zip
gzip / gunzip Compress or decompress a single file. gzip server.log, gunzip server.log.gz
System & Disk Info
Check disk space, RAM and system info. Useful on servers
Command What it does Example
df -h Show disk usage for mounted filesystems in human units. df -h
du -sh Show total size of a directory. du -sh node_modules/
free -h Display free and used memory. free -h
uname -a Show kernel and system information. uname -a
uptime Show how long the system has been running and load averages. uptime
dmesg View kernel ring buffer messages. dmesg | tail
lsof List open files by processes. lsof -i :3000
Users & Sudo Access
Manage logins and privileges. Use sudo carefully
Command What it does Example
whoami Print the current username. whoami
sudo careful Run a command with superuser privileges. sudo systemctl restart nginx
useradd Create a new user account. sudo useradd -m dev
usermod -aG Add user to a group. sudo usermod -aG sudo dev
passwd Change a user password. passwd
chage -l List password expiry information. sudo chage -l dev
Productivity & Misc Quality of Life
Speed up your daily workflow. Use aliases for long cmds
Command What it does Example
alias Create a shortcut for a longer command. alias ll="ls -la"
history Show command history; use !n to re-run. history | tail
man Open manual page for a command. man ssh
cal Display a calendar. cal
watch Run a command repeatedly at intervals. watch -n 5 df -h
Vim: Modes & Basics Vim
Core habits: open, edit, save and quit. Run vimtutor to practice
Keys / Command What it does Mode
vim file.txt Open file in Vim. Shell
i, a, o Enter Insert mode: before cursor, after cursor, or new line below. Normal
Esc Return to Normal mode from any other mode. Any
:w, :q, :wq, :q! Write file, quit, write & quit, or quit without saving. Command
:help Open Vim help system. Command
vimtutor Launch interactive Vim tutorial in the terminal. Shell
Vim: Movement Vim
Navigate without arrow keys. gg top, G bottom
Keys What it does
h, j, k, l Move left, down, up, right.
w, b Jump forward or backward by word.
0, $ Go to start or end of line.
gg, G, 5G Go to first line, last line, or a specific line.
Ctrl+u, Ctrl+d Scroll half-page up or down.
Vim: Editing & Clipboard Vim
Change text and move it around. Think: d=delete, y=yank
Keys What it does
dd, yy, p Delete line, yank (copy) line, paste after cursor.
u, Ctrl+r Undo and redo changes.
ciw Change (delete + insert) the current word.
>>, << Indent or unindent the current line.
v, V, Ctrl+v Visual selection (character, line, block) for operations.
Vim: Search, Replace & Splits Vim
Quickly jump and refactor inside files. :%s/old/new/g to refactor
Keys / Command What it does
/pattern, ?pattern Search forward or backward for a pattern.
n, N Go to next or previous search result.
:%s/old/new/g Replace all occurrences in entire file.
:%s/old/new/gc Replace with confirmation for each match.
:sp file, :vsp file Open file in horizontal or vertical split.
Ctrl+w w Cycle between split windows.
:tabnew file, gt, gT Open new tab and move between tabs.
