1. You open your browser and type:
   http://localhost:3000/

2. Browser action:
   - The browser sends an HTTP GET request to your server.
   - Request looks like:
     GET / HTTP/1.1
     Host: localhost:3000
     User-Agent: Chrome/Edge/Firefox...
     Accept: text/html

3. Express server receives the request:
   - app.listen(PORT) is already running on port 3000.
   - Express checks all routes and middleware in order.

4. Middleware runs first:
   - app.use(cors()) → allows cross-origin requests.
   - app.use(express.json()) → prepares to parse JSON bodies (not needed for GET).
   - app.use(express.static(__dirname)) → checks if a static file matches the request.
     (If you had a file like /style.css, it would serve it directly.)

5. Route matching:
   - The request path is "/".
   - Express finds app.get("/", (req, res) => { ... }).
   - This route matches exactly.

6. Route handler executes:
   res.sendFile(path.join(__dirname, "index.html"));

   - Express builds a response object.
   - It reads the file index.html from your project directory.
   - It sets headers:
     Content-Type: text/html
   - It streams the file contents back to the browser.

7. Browser receives the response:
   - The HTML file is rendered.
   - If index.html contains <script src="script.js"> or <link href="style.css">,
     the browser automatically makes new GET requests for those files.
   - Because of app.use(express.static(__dirname)), Express serves those files too.

8. User sees the homepage.

                    1. User types a gift request in the form and clicks submit.

                    2. Frontend JavaScript runs handleGiftRequest():
                    - Calls fetch("/api/gift-genie", { method: "POST", body: JSON.stringify({ prompt }) })

                    3. Browser sends HTTP POST request:
                    POST /api/gift-genie HTTP/1.1
                    Host: localhost:3000
                    Content-Type: application/json
                    Body: { "prompt": "Gift ideas for my mom" }

                    4. Express receives the request:
                    - app.use(express.json()) parses the JSON body.
                    - req.body = { prompt: "Gift ideas for my mom" }

                    5. Route handler executes:
                    app.post("/api/gift-genie", async (req, res) => { ... })

                    - Extracts prompt from req.body.
                    - Builds messages array.
                    - Calls OpenAI API with messages.

                    6. OpenAI responds with suggestions:
                    Example: "Personalized mug, cozy blanket, scented candles"

                    7. Express sends JSON back:
                    res.json({ reply: "Personalized mug, cozy blanket, scented candles" })

                    - Response headers:
                        Content-Type: application/json
                    - Response body:
                        { "reply": "Personalized mug, cozy blanket, scented candles" }

                    8. Browser frontend receives JSON:
                    const data = await res.json();
                    outputContent.textContent = data.reply;

                    9. User sees the AI’s gift suggestions displayed on the page.

Browser (http://localhost:3000/) 
   ↓ GET /
Express app.get("/") 
   ↓ res.sendFile(index.html)
Browser renders index.html
   ↓ User submits form
Browser fetch("/api/gift-genie", POST JSON)
   ↓ Express app.post("/api/gift-genie")
   ↓ Calls OpenAI API
   ↓ Gets AI response
Express res.json({ reply: "..." })
   ↓ Browser receives JSON
   ↓ Frontend JS updates DOM
User sees gift suggestions
