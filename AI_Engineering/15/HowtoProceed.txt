# 1. Core Concepts (Understand in 5 minutes)

## Authentication vs Authorization

Authentication = Who are you?

Example:

Login with email + password

Authorization = What can you access?

Example:

User A → can see their gift history  
Admin → can see all users

Simple analogy:

| Concept | Example |
|---|---|
| Authentication | Show ID at airport |
| Authorization | Allowed to enter VIP lounge |

# 3. Authentication — Simplest Professional Way

Use JWT authentication.

JWT = JSON Web Token

Flow:

1 User logs in  
2 Server verifies password  
3 Server creates JWT token  
4 Browser stores token  
5 Token sent with every request  

Example request:

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

---
