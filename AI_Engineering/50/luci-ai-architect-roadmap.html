JS Module 6–12: Async, Classes & Patterns Months 2–3 · ~47 hrs ▾

Critical topics (slow down here)

  • Async/Await — you use it, but do you know why it works?
  • Promise chaining, Promise.all, Promise.race, Promise.allSettled
  • The Event Loop — call stack, microtask queue, macrotask queue
  • ES6 Classes, inheritance, private fields, static methods
  • Modules — import/export, dynamic imports, tree-shaking
  • Generators & Iterators — key for LLM streaming
  • Error handling patterns — custom errors, try/catch/finally
  • Notes App + Hacker News full capstone

Why each topic unlocks AI engineering

  • Promise.allSettled → parallel LLM API calls without crashing
  • Generators → streaming tokens from OpenAI in real time
  • Event Loop mastery → no more mystery bugs in async agents
  • Dynamic imports → lazy-load AI models in Cloudflare Workers
  • Custom errors → handle OpenAI rate limits gracefully
⚠ Gate: Don't start the AI course until you can write a custom Promise from scratch and explain microtasks vs. macrotasks. This is the real prerequisite.

Strategic Suggestions
⏱️
Protect your 1–2 hrs ruthlessly
At 1–2 hrs/day, context switching kills you. Pick ONE course, ONE project, ONE resource per week. The biggest mistake is jumping between 4 tutorials. Go deep, not wide.
🗄️
Supabase pgvector is your secret weapon
Most RAG tutorials use local Chroma or FAISS. You're already on Supabase. pgvector in production is rarer than people think — knowing it deeply makes you stand out in any AI engineering interview.
📝
Write an ADR for every project decision
Architecture Decision Records take 15 minutes but prove architect-level thinking. "Why pgvector over Pinecone?" "Why chunking at 512 tokens?" Document it. This becomes your portfolio of thinking, not just code.
🐍
Start Python in Month 3, 30 min/day
JS gets you far in AI engineering. But AI Architects need Python for fine-tuning, evals, and working with the broader ML ecosystem. 30 min/day from Month 3 means you're comfortable by Month 6 without dropping the JS track.
⚡
Cloudflare Workers AI = real differentiator
You're already on Cloudflare Workers & Pages. Workers AI lets you run Llama 3, Whisper, SDXL at the edge — for free. Most AI devs don't know this stack. Build something with it and you're instantly unique.
🎯
The Architect path is about decisions, not code
Tech Leads/Architects spend 60% of their time thinking about trade-offs, not writing code. Practice this from Month 4: for every feature, write "Option A vs B vs C — I chose A because…". That habit is the career leap.
