## 📐 Image Size & Orientation
- **DALL·E 2**:
  - Default: `1024 × 1024` (square)
  - Options: `256 × 256` (small), `512 × 512` (medium)

- **DALL·E 3**:
  - Default: `1024 × 1024` (square)
  - Portrait: `1024 × 1792`
  - Landscape: `1792 × 1024`

👉 Example:  
*“Steampunk city in a canyon, portrait orientation”* → `size=1024x1792`  
*“Steampunk city in a canyon, landscape orientation”* → `size=1792x1024`

---

## 🌟 Image Quality
- **Parameter:** `quality`
- Options:
  - `standard` → faster, cheaper, lower detail
  - `hd` → slower, more expensive, higher detail

👉 Example:  
*“Steampunk city in a canyon, quality=hd”* → sharper textures, finer details, more realistic lighting.

⚠️ Note: HD images cost nearly double compared to standard.

---

## 🎨 Image Style
- **Parameter:** `style` (DALL·E 3 only)
- Options:
  - `vivid` (default) → hyper‑real, dramatic, colorful
  - `natural` → photo‑realistic, simpler, camera‑like

👉 Example:  
- `style=vivid`: *“Epic canyon city glowing with neon gas lamps.”*  
- `style=natural`: *“Photo of a golden brown espresso shot in a ceramic cup.”*

---

## 🖼️ Practical Use Cases
- **Vivid style** → fantasy art, surreal landscapes, dramatic posters.  
- **Natural style** → product mockups, logos, stickers, realistic photos.  
- **HD quality** → when you need fine textures (metal, fabric, reflections).  
- **Size adjustments** → portrait for vertical posters, landscape for wide scenic shots, square for stickers or icons.

## 🎯 Your Takeaway
- **Size** → controls orientation (square, portrait, landscape).  
- **Quality** → standard vs HD (trade‑off: speed/cost vs detail).  
- **Style** → vivid vs natural (dramatic vs photo‑realistic).  

            ## 🖼️ Basic Image Generation (Square Default)
            ```javascript
            import OpenAI from "openai";

            const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

            async function generateSquare() {
            const response = await openai.images.generate({
                model: "dall-e-3",
                prompt: "A futuristic city floating in the clouds of Venus",
                size: "1024x1024" // square default
            });

            console.log(response.data[0].url);
            }

            generateSquare();
            ```
