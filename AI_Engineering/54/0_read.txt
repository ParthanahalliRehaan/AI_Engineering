## 🛠 Setting up Transformers.js in the Browser

We’re using the **transformers.js** library, which lets us run Hugging Face models directly in the browser — no API calls needed once the model is downloaded.

### 1. Import the pipeline
```javascript
import { pipeline } from '@xenova/transformers';

// Create an object detection pipeline
const detector = await pipeline('object-detection', 'Xenova/detr-resnet-50');
```

- `pipeline` is the helper that loads the model and tokenizer.
- We specify the **task** (`object-detection`) and the **model name** (`Xenova/detr-resnet-50`).
- This is asynchronous, so we `await` it.

---

### 2. Status Indicator
Since loading can take time, we update the UI:
```javascript
document.getElementById('status').innerText = 'Loading model...';

const detector = await pipeline('object-detection', 'Xenova/detr-resnet-50');

document.getElementById('status').innerText = 'Ready!';
```

This way, users know the app isn’t frozen.

---

### 3. Detect Objects in an Image
```javascript
const imageUrl = 'road.jpg'; // your demo image

const results = await detector(imageUrl, {
  threshold: 0.95,   // model must be 95% confident
  percentage: true   // return coordinates in percentages
});

console.log(results);
```

- Each result contains:
  - `label`: the object name (e.g., "car", "person")
  - `score`: confidence level
  - `box`: coordinates (`x`, `y`, `width`, `height`)

---

### 4. Drawing Boxes on the Image
We already have a helper function `drawObjectBox`. Now we loop through results:

```javascript
function detectObjects() {
  detector('road.jpg', { threshold: 0.95, percentage: true })
    .then(objects => {
      objects.forEach(obj => {
        drawObjectBox(obj); // helper handles the UI
      });
    });
}
```

This connects the AI output to the UI.
